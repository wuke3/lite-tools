  • 群聊

    白名单模式：在这些群里才抢；黑名单模式：在这些群里不抢

  • 发送者

    白名单模式：这些发送者的红包才抢；黑名单模式：这些发送者的红包不抢

    抢到红包后通知方式

  • 通知目标

    自己
    我的手机
    QQ好友
    群聊

  • 通知对象

    QQ好友或群聊的ID

    抢到红包后的通知内容，%peerName%=群名，%peerUid%=群号，%senderName%=发送者，%sendUin%=发送者QQ，%amount%=金额
